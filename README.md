# Linkdown

Convert any webpage into **Markdown** or **PDF** instantly.

Paste a URL, click convert, and download a clean document ready for notes, archiving, or documentation.

Built for speed and simplicity.

---

## Features

• Convert webpages to **Markdown**  
• Export pages as **PDF**  
• Minimal interface — paste URL and go  
• Works with most articles and blogs  
• Deployable on Netlify in seconds
